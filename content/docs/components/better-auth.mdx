## Features

- **SQLite Database** - Uses SQLite for simple, file-based data storage
- **TypeScript Support** - Full TypeScript support out of the box
- **Next.js Integration** - Seamlessly integrates with Next.js App Router
- **Zero Configuration** - Works with minimal setup required

## Basic Example

```tsx
import { authClient } from "@/lib/auth-client";

export function LoginButton() {
  const handleLogin = async () => {
    await authClient.signIn.email({
      email: "user@example.com",
      password: "password",
    });
  };

  return <button onClick={handleLogin}>Sign In</button>;
}
```

{/* ## Database

This setup uses SQLite which creates a `sqlite.db` file in your project root. Better Auth will automatically handle database migrations and table creation when you first run your application. */}
